# Chess V1

A chess app where you can play against bots with personalities, solve puzzles, and learn chess concepts.

## Running it locally

You'll need [Node.js](https://nodejs.org) installed (version 18 or higher).

```bash
git clone https://github.com/sarp-e/chess-v1.git
cd chess-v1
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.
